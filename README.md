# django-vue3-exercise
An exercise with django + vue3

## Backend - Django

```bash
$ cd backend
$ pip install -r ../requirements.txt
$ python manage.py runserver
```
Models:
- `accounts`
- `catalog`

## Frontend - Vue.js
```bash
$ cd frontend
$ npm install 
$ npm run dev
```
